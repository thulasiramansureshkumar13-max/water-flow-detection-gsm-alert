# Water Flow Detection and GSM Alert System

Embedded system to detect abnormal water flow conditions and dispatch
real-time SMS alerts via GSM module using AT commands. Features
event-driven firmware and automated sensor data acquisition.

## Features
- Real-time abnormal flow condition detection
- GSM SMS alert using AT commands
- Event-driven embedded firmware logic
- Automated sensor data acquisition
- Low-latency alert dispatch system

## Hardware
- PIC / STM32 microcontroller
- Flow sensor
- GSM module (SIM800L / SIM900)
- IDE: MPLAB X IDE / STM32CubeIDE

## How It Works
1. Flow sensor continuously monitors water flow rate
2. Firmware detects abnormal flow condition
3. GSM module is triggered via AT commands
4. SMS alert is sent to registered number in real time

## Applications
- Smart water management systems
- Industrial pipeline monitoring
- IoT-based infrastructure alert systems
